## Backend için 

1. Terminalinizi açın.
2. `BlogApi` dizinine gidin:

   ```bash
   cd BlogApi

## Projeyi Başlatmak için 

   ```bash
   npm run start
   ```

## Frontend için 
1. Terminalinizi açın.
2. `BlogFrontend` dizinine gidin:

   ```bash
   cd BlogFrontend

## Projeyi Başlatmak için 

   ```bash
   npm run dev
   ```
